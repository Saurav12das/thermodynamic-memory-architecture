### General V2 Architecture Recommendations

1. **Embedding-based retrieval**: Replace keyword search in R0 with vector embeddings for semantic matching. This would significantly improve recall on D0 and D4 questions.
2. **LLM-backed verification gate**: The structural verification gate (temporal + source + contradiction) is a good foundation but needs an LLM verifier for nuanced fact-checking.
3. **Active consolidation**: Trigger consolidation automatically when the system reaches REFLECTIVE stage, not just on manual calls.
4. **Forgetting curve**: Implement explicit forgetting for L0_buffer memories that don't get promoted within a time window.
5. **Cross-layer retrieval**: Currently retrieval searches within layers. Add cross-layer query fan-out for D4 abstraction questions.
6. **Feedback loop to jolt encoder**: Use downstream task outcomes to adjust jolt thresholds dynamically, not just domain experience count.
